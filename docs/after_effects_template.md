## 🎬 制作流程

### 1. 创建基础形状（30秒）

#### 眼白（巩膜）
```
工具: 椭圆工具
大小: 120x120 像素
颜色: 白色 (#FFFFFF)
位置: (60, 120) 和 (180, 120) - 左右眼
```

#### 虹膜
```
工具: 椭圆工具
大小: 60x60 像素
颜色: 蓝色/绿色/棕色
位置: 眼白中心
```

#### 瞳孔
```
工具: 椭圆工具
大小: 30x30 像素
颜色: 黑色 (#000000)
位置: 虹膜中心
```

### 2. 创建动画（2分钟）

#### 示例 1：开心表情

**效果描述：** 眼睛变成弯月形，有星星闪烁

**关键帧设置：**
```
0:00 - 正常眼睛
0:15 - 眼睛开始变窄（缩小 Scale Y）
0:30 - 眼睛变成弯月形（最小 Scale Y）
1:00 - 保持弯月形
2:00 - 眼睛开始恢复
2:30 - 恢复正常
3:00 - 结束
```

**具体操作：**
1. 选中"虹膜"和"瞳孔"图层
2. 按 S 键打开 Scale 属性
3. 在 0:15 设置关键帧：Scale = 100%
4. 在 0:30 设置关键帧：Scale = (100%, 10%) - Y轴压缩
5. 在 2:00 设置关键帧：Scale = (100%, 10%)
6. 在 2:30 设置关键帧：Scale = 100%
7. 选中所有关键帧，按 F9 添加缓动

#### 示例 2：惊讶表情

**效果描述：** 眼睛放大，瞳孔收缩

**关键帧设置：**
```
0:00 - 正常眼睛
0:10 - 眼睛快速放大 (Scale: 100% → 150%)
0:15 - 瞳孔收缩 (Scale: 100% → 50%)
0:30 - 保持惊讶状态
2:00 - 开始恢复
2:30 - 恢复正常
```

**具体操作：**
1. 选中"眼白"图层
2. Scale: 0:10 = 100%, 0:15 = 150%
3. 选中"瞳孔"图层
4. Scale: 0:15 = 100%, 0:20 = 50%

### 3. 添加特效（1分钟）

#### 星星闪烁（开心表情）
1. 新建形状图层 "Star"
2. 使用多边形星形工具绘制星星
3. 位置：眼睛上方
4. 动画：
   - 0:00 - Opacity: 0%
   - 0:30 - Opacity: 100%
   - 1:00 - Opacity: 0%
   - 循环：使用表达式 `loopOut()`

#### 泪滴（伤心表情）
1. 新建形状图层 "Tear"
2. 绘制水滴形状
3. 位置：从眼睛底部开始
4. 动画：
   - Position: 从眼睛底部向下移动

### 4. 优化导出设置（1分钟）

#### 添加到渲染队列
1. **Composition** → **Add to Adobe Media Encoder Queue**
2. 在 Media Encoder 中：
   - **Preset**: 选择 "Match Source - Adjusted for 240x240"
   - **Format**: PNG
   - **Color Mode**: RGB (如果需要透明，选择 RGBA + Transparency)
   - **Resolution**: 240x240

#### 渲染设置
```
输出位置: ./animations/happy/
文件命名: happy_####.png (会自动生成 happy_0000.png 等)
```

## 🎭 表情模板预设

### 1. Happy（开心）
```
特征:
- 眼睛变成弯月形 (Scale Y: 10%)
- 星星在上方闪烁
- 可选：添加 ^_^ 符号

时长: 2-3 秒
```

### 2. Surprised（惊讶）
```
特征:
- 眼白放大 150%
- 虹膜/瞳孔保持原大小或稍放大
- 快速变化（10-15 帧）

时长: 1-2 秒
```

### 3. Sad（伤心）
```
特征:
- 眼睛角度倾斜 15°（Rotation）
- 眼睑下垂（使用遮罩或形状）
- 泪滴从眼角落下
- 灰蓝色调（可选）

时长: 3-4 秒
```

### 4. Angry（生气）
```
特征:
- 添加眉毛形状（倒 V 形）
- 眉毛向下压
- 眼睛变红（可选，Tint 效果）
- 快速抖动（可选，Wiggle 表达式）

时长: 2-3 秒
```

### 5. Thinking（思考）
```
特征:
- 眼睛左右移动
- 添加 "??" 符号
- 可选：添加灯泡图标

时长: 循环动画，2-3 秒循环
```

### 6. Idle（待机）- 循环动画
```
特征:
- 轻微眨眼（每 2-3 秒一次）
- 瞳孔缓慢移动（Ping Ppong 动画）
- 整体保持静止但自然

时长: 3-5 秒循环
使用表达式: loopOut("pingpong")
```

### 7. Talking（说话）
```
特征:
- 眼睛随语音节奏轻微缩放
- 可以参考音频波形生成关键帧

时长: 循环动画
```

## 🎬 高级技巧

### 使用表达式创建循环动画

#### 待机动画循环
```javascript
// 在 Position 属性上
// 瞳孔缓慢左右移动
x = 60 + Math.sin(time * 2) * 10;
y = 120;
[x, y]

// 在 Scale 属性上
// 轻微呼吸效果
s = 100 + Math.sin(time * 3) * 2;
[s, s]
```

#### 眨眼循环
```javascript
// 在 Scale Y 属性上
// 每 3 秒眨眼一次
if (time % 3 < 0.1) {
    10  // 闭眼
} else {
    100 // 睁眼
}
```

### 使用 Graph Editor 优化动画

1. 选中关键帧
2. **Window** → **Graph Editor**
3. 调整曲线让动画更自然：
   - 进入：慢速
   - 中间：快速
   - 离开：慢速

### 使用预设（Presets）

保存常用动画为预设：
1. 选中所有关键帧
2. **Animation** → **Save Animation Preset**
3. 命名为 "Happy Eye Animation"
4. 下次直接应用到新合成

## 📤 导出检查清单

导出前确保：
- [ ] 分辨率是 240x240
- [ ] 帧率是 12 fps
- [ ] 时长不超过 5 秒（60 帧）
- [ ] 背景是黑色或透明
- [ ] 所有图层都在画面内
- [ ] 动画流畅（预览一遍）
- [ ] 文件名按顺序编号

## 🔄 批量导出多个表情

使用渲染队列批量处理：
1. 为每个表情创建单独的合成
2. 全选所有合成
3. **Composition** → **Add to Adobe Media Encoder Queue**
4. Media Encoder 会自动排队渲染
