# 表情切换快速指南

## 🎭 通过语音切换动画

### 简单命令（推荐）

直接说以下任意一句话：

```
"换下一个动画"
"换下一个眼睛"
"切换动画"
"换一个表情"
"下一个"
```

机器人会自动在两个动画之间循环切换。

### 指定动画名称

```
"切换到 Grok 动画"
"播放 Grok"
"换成 Grok 表情"
"切换到默认眼睛"
"使用默认动画"
"切换到 eye 动画"
```

---

## 📝 可用的动画

| 动画名称 | 中文描述 | 英文关键词 |
|---------|---------|-----------|
| **默认眼睛** | 友好、平静的眼睛 | eye, default |
| **Grok** | 有趣、生动的 Grok 动画 | grok |

---

## 🔄 自动切换

使用"下一个"命令会在两个动画之间循环切换：

```
默认眼睛 ←→ Grok
```

每次说"换下一个"都会切换到另一个动画。

---

## 💡 使用技巧

### ✅ 推荐的说法

- "换下一个动画"
- "切换到 Grok"
- "换成默认眼睛"
- "下一个"

### ❌ 不推荐的说法

- "G表情"（AI 听不懂）
- "g眼"（AI 听不懂）
- "G 动画"（AI 可能误解）

### ✅ 正确的说法

- 使用完整的英文名："Grok"
- 或使用中文描述："下一个动画"

---

## 📸 拍照识别优化

当你说"帮我看一下这是什么"时，机器人会先回应：

```
"好的，稍等我帮你看看"
"好的，我来看看这是什么"
"没问题，我来帮你识别一下"
```

然后再拍照并分析，让你知道它在工作。

## 🔧 技术信息

### 动画规格
- **尺寸**: 240x240 全屏
- **帧率**: 8 FPS
- **循环**: 自动循环播放
- **切换时间**: < 50ms（几乎即时）

### 可用命令

| 命令 | 功能 |
|------|------|
| `"next"` | 切换到下一个动画 |
| `"previous"` | 切换到上一个动画 |
| `"eye"` / `"default"` | 默认眼睛动画 |
| `"grok"` | Grok 动画 |

## ❓ 常见问题

### Q: AI 听不懂 "Grok" 怎么办？
A: 使用完整清晰的说法："切换到 Grok 动画" 或 "换成 Grok 表情"

### Q: 如何知道当前是哪个动画？
A: 看屏幕显示。默认眼睛比较平静，Grok 动画更生动。

### Q: 可以添加更多动画吗？
A: 可以！参考 [animation_system_complete_guide.md](animation_system_complete_guide.md) 中的"添加新表情"章节。
